Problem

Surveillance is an integral part of security and control. The job requires extended periods of looking into multiple video footages for an undesirable incident or evidence following the incident, which proves to be highly inefficient considering the massive amount of footages. This manual task provides evidence in court but is rarely used to prevent crime or react to it in real-time.

Read More

Our Solution

Developing a real-time crime detection technology that can be integrated with any security system, to ensure public safety through visual crowd surveillance.

About

Our Motive

The main motive behind building this project includes:
  • Narrow down the Crime rates and ensure that law is enforced.
  • Decreasing the assaults taking place on a daily basis thereby providing a safe environment to women and children so that they can feel a sense of security outside of their homes too.
  • Reduce the labor and toil of human effort and automate the whole process.

The motive behind our idea is to reduce the crime and assaults trudging on the road on a daily basis which goes undetected. We aim to reduce the lengthy and tedious process of observing each CCTV Footage and rather make the whole process in a real time frame so that the deed is detected before its too late!

TECH STACK

Machine & Deep Learning

We used TensorFlow that is an end-to-end open source platform for machine learning. We used comprehensive and flexible tools, libraries and high-level APIs like Keras to build and train our ML model.

OPEN-CV

OpenCV (Open Source Computer Vision Library) is an open source computer vision library which was used to detect activities in a crowded area and narrow down the region of search.

WEB DEVELOPMENT

To increase the effectiveness of our presentation we developed a mini-website using HTML,CSS and Javascript and used django framework in our backend to make the site responsive, which was then hosted on Amazon Web Services.
